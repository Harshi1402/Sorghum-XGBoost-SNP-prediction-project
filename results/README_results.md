# Results

This folder contains visual outputs and summary tables generated during the project.  
No raw SNP data or identifiers are included.

## Included (Portfolio-Safe)

- Training predicted vs actual plot
- Validation predicted vs actual plot
- Feature importance barplot (gain)
- Gain vs frequency scatterplot
- Summary table of feature importance (IDs removed)

## Not Included

- Raw VCF files
- Phenotype data
- SNP identifiers
- Full feature matrices

These exclusions ensure ethical handling of genomic data and compliance with academic integrity.
